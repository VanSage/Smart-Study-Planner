# Smart Study Planner

Smart Study Planner is a simple and responsive website designed to help students organize their study schedules and improve productivity through an easy-to-use interface.

## Features

* Responsive design
* Study planning interface
* Simple and clean layout
* User-friendly structure

## Technologies Used

* HTML

## Project Purpose

The project was developed to create a simple web-based study planner focused on organization and productivity.
